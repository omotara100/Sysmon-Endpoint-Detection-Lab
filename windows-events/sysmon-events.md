# Sysmon Event Analysis

## Objective
Analyze endpoint behavior using Sysmon telemetry.

## Key Event IDs
- Event ID 1 – Process Creation
- Event ID 3 – Network Connection
- Event ID 10 – Process Injection
- Event ID 22 – DNS Queries

## Notable Findings
- Command-line execution observed
- Non-browser processes making outbound connections

## Analyst Notes
Sysmon enables behavior-based detection rather than signature-only detection.
