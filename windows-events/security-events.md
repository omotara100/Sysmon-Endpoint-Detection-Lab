# Windows Security Event Analysis

## Objective
Analyze authentication-related activity from the Windows Security log.

## Key Event IDs
- 4624 – Successful logon
- 4625 – Failed logon
- 4672 – Special privileges assigned

## Observations
- Normal user logons observed
- Failed logon attempts may indicate brute-force or misconfiguration

## Analyst Notes
Security logs provide critical context for lateral movement and credential abuse.
